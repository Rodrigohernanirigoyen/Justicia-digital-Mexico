**Incremento 2026:** 3.69% respecto a UMA 2025 ($113.14).

**Nota:** La UMA se usa para multas, límites de créditos Infonavit, cotizaciones al IMSS (tope de 25 UMAs), y exenciones de ISR. NO se usa para calcular prestaciones laborales (esas se calculan con el salario real del trabajador).

## Salario Base de Cotización (SBC) mínimo

```
SBC mínimo 2026 = Salario mínimo × Factor de integración (primer año, prestaciones mínimas)
SBC mínimo 2026 = $315.04 × 1.0493 = $330.57 diarios
```

Este es el SBC más bajo que un patrón puede reportar al IMSS para un trabajador de tiempo completo en zona general. Reportar un SBC inferior es infracción sancionable.

```
Factor = 1 + (Aguinaldo / 365) + (Vacaciones × Prima vacacional / 365)

```

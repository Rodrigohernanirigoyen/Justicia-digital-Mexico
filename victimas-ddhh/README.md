# Víctimas DDHH — Plugin para víctimas de delitos y violaciones a derechos humanos en México

Plugin de Claude para orientar a víctimas y sus familias sobre sus derechos, las instituciones que pueden ayudarles, y los pasos concretos para acceder a la verdad, la justicia y la reparación integral. Diseñado con enfoque de trauma informado.

## Para quién es

Para cualquier persona en México que haya sido víctima de un delito o de una violación a sus derechos humanos, o que tenga un familiar en esa situación. Especialmente:

- Familiares de personas desaparecidas.
- Víctimas de violencia de género (feminicidio, violencia familiar, abuso sexual, trata).
- Víctimas de tortura, detención arbitraria, ejecución extrajudicial.
- Personas desplazadas forzosamente.
- Defensores de derechos humanos y periodistas en riesgo.
- Colectivos de búsqueda y OSCs que acompañan víctimas.
- Abogados y clínicas jurídicas que litigan casos de DDHH.

## Qué hace

| Comando | Para qué |
|---|---|
| `/victimas-ddhh:cold-start-interview` | Configuración inicial |
| `/victimas-ddhh:que-hago-si` | Orientación de primer contacto según la situación |
| `/victimas-ddhh:desaparicion-forzada` | Ruta completa para casos de desaparición |
| `/victimas-ddhh:violencia-genero` | Ruta para violencia de género |
| `/victimas-ddhh:registro-victimas` | Inscripción al RENAVI / registro estatal |
| `/victimas-ddhh:denuncia-penal` | Cómo presentar denuncia o querella |
| `/victimas-ddhh:reparacion-integral` | Derechos de reparación: restitución, rehabilitación, compensación, satisfacción, garantías de no repetición |
| `/victimas-ddhh:amparo-victimas` | Amparo como herramienta de protección de derechos de víctimas |
| `/victimas-ddhh:medidas-proteccion` | Medidas de protección y órdenes de protección |

## Enfoque de trauma informado

Este plugin opera bajo principios de atención centrada en la víctima:

- **Seguridad primero.** Si hay riesgo activo, las primeras acciones son de protección, no de orientación legal.
- **Sin revictimización.** No se piden detalles innecesarios del hecho victimizante. Se pregunta lo mínimo para orientar.
- **Autonomía.** La víctima decide qué hacer. El plugin orienta, no presiona.
- **Lenguaje cuidadoso.** Sin minimizar, sin dramatizar, sin culpar.
- **Conexión con redes de apoyo.** Siempre referir a acompañamiento psicológico además de legal.

## Advertencia importante

Este plugin es informativo y orientativo. **No sustituye** el acompañamiento legal y psicológico profesional que las víctimas necesitan y merecen. Las rutas legales descritas son complejas y pueden variar según el caso, la jurisdicción, y las circunstancias. Siempre referir a CEAV, comisiones estatales, CNDH, y/o OSCs especializadas.
